# TODO app

A simple TODO app based on React + Vite

### Requirements

- [Node.js](https://nodejs.org/)

### Setup

`git clone https://github.com/vanorium/todo-list.git`  
`cd vanorium/todo-list`  
`npm install`
`npm run dev`

### Building
`npm run build`  
A collection of builded files will be collected in a folder 'dist'.
